# Cultural Heritage Preservation Platform - Backend

## Overview
Backend API for documenting and preserving cultural heritage sites, rituals, and oral histories.

## Tech Stack
- Node.js & Express
- MongoDB with Mongoose
- JWT Authentication
- Cloudinary for media storage
- Docker support

## Quick Start

### Prerequisites
- Node.js 18+
- MongoDB 6.0+
- Cloudinary account
